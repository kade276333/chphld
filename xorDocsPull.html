Chemp Physics Lua API Reference

Documentation for the XORWIRE Lua modding API.

LuaAPI

GetTransformByName

Returns a TransformWrapper for a GameObject by name. Returns null if not found.

local playerTransform = API.GetTransformByName("Player")

GetObject

Returns a GameObjectWrapper for a GameObject by name. Returns null if not found.

local enemy = API.GetObject("Enemy")

DebugLog

Logs a message to the Unity console.

API.DebugLog("Hello from Lua!")

ChempDebuggerLog

Logs a message to the ChempDebugger system.

API.ChempDebuggerLog("Debug info")

GetGripValue / GetTriggerValue

Returns the analog grip/trigger input value (0–1) for the left or right hand.

local grip = API.GetGripValue(true)
local trigger = API.GetTriggerValue(false)

GetPrimaryButton / GetSecondaryButton

Returns whether the primary or secondary button is pressed for left/right hand.

if API.GetPrimaryButton(true) then
    -- do something
end
if API.GetSecondaryButton(false) then
    -- do something
end

SendHaptic

Sends haptic feedback to a controller.

API.SendHaptic(true, 0.2, 0.5)

SetActive / DestroyObject / SetTag

Activate, destroy, or set the tag of a GameObjectWrapper.

local obj = API.GetObject("Platform")
API.SetActive(true, obj)
API.SetTag("Interactable", obj)
API.DestroyObject(obj)

GetObjectByTag

Returns a list of GameObjectWrapper objects with the specified tag.

local enemies = API.GetObjectByTag("Enemy")
for i, enemy in ipairs(enemies) do
    enemy:GetTransform():SetPosition(0, 1, 0)
end

GetPhysRig / GetMainCamera

Returns the physics rig or main camera as a GameObjectWrapper.

local rig = API.GetPhysRig()
local cam = API.GetMainCamera()

CreatePrimitiveObject

Creates a primitive object (Cube, Sphere, etc.) and returns a GameObjectWrapper.

local cube = API.CreatePrimitiveObject("Cube")
cube:GetTransform():SetPosition(0, 2, 0)

GetRigidbody / AddRigidbody

Get or add a RigidbodyWrapper to a GameObjectWrapper.

local rb = API.GetRigidbody(obj)
local rb2 = API.AddRigidbody(obj)

RigidbodyWrapper Methods

Methods for manipulating rigidbodies.

rb:SetVelocity(0, 10, 0)
rb:AddForce(0, 5, 0)
rb:SetKinematic(true)
rb:SetUseGravity(false)
rb:MakeGrabbable(true)

TransformWrapper

Wraps a Transform and allows getting/setting position, rotation, scale, and LookAt.

local pos = transform:GetPos()
transform:SetPosition(1, 2, 3)
transform:LookAt(target)

GameObjectWrapper

Wraps a GameObject and allows accessing its Transform and name.

local t = obj:GetTransform()
local name = obj:GetName()
obj:SetName("NewName")

Vector3Wrapper

Wraps a Vector3 for easy get/set of X, Y, Z and conversion to Unity Vector3.

local x = vec:GetX()
vec:SetY(5)
local v3 = vec:ToVector3()

RigidbodyWrapper

Wraps a Rigidbody for Lua access.

rb:SetVelocity(1,0,0)
rb:AddForce(0,5,0)
rb:SetKinematic(true)
rb:SetUseGravity(false)
